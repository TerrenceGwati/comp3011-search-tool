# COMP3011 Coursework 2 – Search Engine Tool

A command-line search tool that crawls [quotes.toscrape.com](https://quotes.toscrape.com), builds an inverted index, and lets you search for words across all crawled pages.

## Project Overview

This tool has three main components:
- **Crawler** (`crawler.py`) – visits every page on the site following internal links, with a 6-second politeness delay between requests
- **Indexer** (`indexer.py`) – builds an inverted index storing each word's frequency and positions per page, saved as JSON
- **Search** (`search.py`) – takes user queries and returns ranked results using combined frequency scoring

## Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd <repo-name>
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the tool from the `src/` directory:

```bash
cd src
python main.py
```

### Commands

| Command | Description | Example |
|---|---|---|
| `build` | Crawl the site and build the index | `> build` |
| `load` | Load a previously saved index | `> load` |
| `print <word>` | Show the index entry for a word | `> print nonsense` |
| `find <query>` | Find pages containing the word(s) | `> find good friends` |
| `help` | Show available commands | `> help` |
| `exit` | Exit the program | `> exit` |

### Example Session

## Testing

Run all tests from the project root:

```bash
python -m pytest tests/ -v
```

Or run individual test files:

```bash
python -m pytest tests/test_crawler.py -v
python -m pytest tests/test_indexer.py -v
python -m pytest tests/test_search.py -v
```
